# AttSiOff

A self-attention-based approach on siRNA design with inhibition and off-target effect prediction

Here is the codes and dataset for inhibition predictor.
1. folder "attention_2_mse_loss" contains model parameters and figures during 10-fold cross validation and cross-dataset validation.
2. for pre-trained RNA-FM embedding of antisense strand and local mrna sequence, please download from https://pan.baidu.com/s/1pJmvxY5wBSGzTf-v8KXxOQ (pwd: gcjn)
